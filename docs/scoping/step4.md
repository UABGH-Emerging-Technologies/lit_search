# Step 4: Summarization

Summarizes articles within each category using a chunked map-reduce strategy.
Produces a per-category narrative with APA-style in-text citations.

**Input:** Categorized article DataFrame with full text

**Output:** Category summaries (base64-encoded DOCX)
